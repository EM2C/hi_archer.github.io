【NOIP 2014】解方程 hash+秦九韶

发表于 2018-09-02 | 分类于 数论

3732 解方程
CODEVS
题目描述 Description

输入描述 Input Description
输入文件名为equation.in。

输入共n+2行。

第一行包含2个整数n、m，每两个整数之间用一个空格隔开。

接下来的n+1行每行包含一个整数，依次为a0,a1,a2,……,an。

输出描述 Output Description
输出文件名为equation.out。

第一行输出方程在[1, m]内的整数解的个数。

接下来每行一个整数，按照从小到大的顺序依次输出方程在[1, m]内的一个整数解。

题解：
可以用秦九韶加速运算，然后取一定的质数然后看是否对于所有的质数这个X都为0，若是这个很有可能就是答案（确实我们就是把这个值作为答案的）。恩就是这样，我一直在写挂，不知道为什么。。。。重新写了好多次
